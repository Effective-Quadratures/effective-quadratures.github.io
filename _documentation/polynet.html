Deep learning with polynomials¶

Deep learning with polynomials using polynomial activation functions.

class equadratures.polynet.Polynet(sample_points, sample_outputs, num_ridges, max_iters=1, learning_rate=0.001, W=None, coeffs=None, momentum_rate=0.001, opt='sd', poly_deg=2, verbose=False)[source]¶

Definition of a polynomial neural network framework.

Parameters

  • sample_points (numpy.ndarray) – Sample training data inputs.

  • sample_outputs (numpy.ndarray) – Sample training data outputs.

  • num_ridges (int) – The number of ridges used in the neural network/

  • max_iters (int) – The maximum number of iterations for the gradient descent optimisation algorithm.

  • learning_rate (double) – The learning rate used in the optimisation.

  • W (numpy.ndarray) – An initial ridge for the optimisation algorithm.

  • coeffs (numpy.ndarray) – Coefficients for the polynomial 1D-ridge functions.

  • opt (string) – The optimisation strategy used. Options include: sd for steepest descent, mom for a momentum-rate-based approach or adapt for an adaptive learning rate based approach.

  • momentum_rate (double) – The momentum rate used.

  • poly_deg (int) – The degree of the polynomials used throughout the neural network.

  • verbose (bool) – A verbose flag.

Sample constructor initialisations:

net = Polynet(X,Y,num_ridges,max_iters=20000, learning_rate=1e-4, momentum_rate=.001, opt='adapt')
net.fit()
