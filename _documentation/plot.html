Plotting utilities¶

Plotting utilities.

equadratures.plot.plot_2D_contour_zonotope(mysubspace, minmax=[- 3.5, 3.5], grid_pts=180, save=False, xlabel='$u_1$', ylabel='$u_2$', xlim=None, ylim=None, show=True, return_figure=False)[source]¶

Generates a 2D contour plot of the polynomial ridge approximation.

Parameters

  • mysubspace (Subspace) – An instance of the Subspace class.

  • minmax (list) – A list of the minimum and maximum values of \(M^T x\), where \(M\) is the subspace.

  • grid_pts (int) – The number of grid points for generating the contour plot.

  • save (bool) – Option to save the plot as a .png file.

  • xlabel (string) – The label used on the horizontal axis.

  • ylabel (string) – The label used on the vertical axis.

  • xlim (list) – Lower and upper bounds for the horizontal axis, for example xlim=[-3, 5].

  • ylim (list) – Lower and upper bounds for the vertical axis, for example ylim=[-1, 1].

  • show (bool) – Option to view the plot.

  • return_figure (bool) – Option to return the figure and axes instances of the matplotlib classes.

equadratures.plot.plot_orthogonal_polynomials(Parameter, ax=None, order_limit=None, number_of_points=200, save=False, xlim=None, ylim=None, show=True, return_figure=False)[source]¶

Plots the first few orthogonal polynomials.

Parameters

  • Parameter (Parameter) – An instance of the Parameter class.

  • ax (matplotlib.ax) – An instance of the matplotlib axes class.

  • order_limit (int) – The maximum number of orthogonal polynomials that need to be plotted.

  • number_of_points (int) – The number of points used for plotting.

  • save (bool) – Option to save the plot as a .png file.

  • xlim (list) – Lower and upper bounds for the horizontal axis, for example xlim=[-3, 5].

  • ylim (list) – Lower and upper bounds for the vertical axis, for example ylim=[-1, 1].

  • show (bool) – Option to view the plot.

  • return_figure (bool) – Option to return the figure and axes instances of the matplotlib classes.

Example:

import numpy as np
from equadratures import *

myparam = eq.Parameter(distribution='uniform', lower = -1.0, upper = 1.0, order=8, endpoints='both')
myparam.plot_orthogonal_polynomials(xlim=[-1.6, 1.6])
